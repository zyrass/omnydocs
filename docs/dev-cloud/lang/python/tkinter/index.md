## Présentation

!!! quote "Analogie pédagogique — La Boîte à Outils IKEA"
    Tkinter est la boîte à outils bureau fournie avec Python : tout est déjà là, aucune installation supplémentaire. Vous assemblez des widgets (boutons, labels, champs texte) comme des meubles IKEA — chaque pièce a son rôle, et les instructions (gestionnaires d'événements) décrivent comment tout s'assemble. Ce n'est pas le style le plus moderne, mais c'est rapide, portable et suffisant pour la majorité des outils internes.

**Tkinter** est la bibliothèque GUI standard de Python. Elle est incluse dans toutes les installations Python et permet de créer des interfaces desktop multiplateformes (Windows, macOS, Linux) sans aucune dépendance externe.

| Caractéristique | Valeur |
|---|---|
| **Inclus dans** | Python standard library |
| **Widgets** | Button, Label, Entry, Canvas, Treeview... |
| **Layout** | pack, grid, place |
| **Cas d'usage** | Outils internes, prototypes, scripts GUI |
