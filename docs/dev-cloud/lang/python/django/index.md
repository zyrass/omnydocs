## Présentation

!!! quote "Analogie pédagogique — La Cuisine Prête à l'Emploi"
    Django, c'est comme une cuisine professionnelle entièrement équipée : le four (ORM), le plan de travail (templates), le garde-manger (admin), la caisse (authentification) sont déjà installés. Vous n'avez plus qu'à préparer votre recette (l'application). À l'inverse de Flask — qui vous donne juste une plaque chauffante — Django vous offre le restaurant complet.

**Django** est le framework web Python le plus complet. Il suit le principe **"batteries included"** : ORM, admin auto-généré, authentification, formulaires, migrations et sécurité sont fournis dès l'installation.

| Caractéristique | Valeur |
|---|---|
| **Créé par** | Adrian Holovaty & Simon Willison (2005) |
| **Philosophie** | Convention over Configuration |
| **ORM** | Intégré (QuerySet API) |
| **Admin** | Auto-généré depuis les modèles |
| **Cas d'usage** | SaaS, CMS, API REST, e-commerce |
