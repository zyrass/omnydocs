## Présentation

!!! quote "Analogie pédagogique — Le Plan de Ville"
    Une ville sans plan d'urbanisme finit par être un chaos de routes, d'égouts et de bâtiments qui s'entremêlent. L'architecture logicielle est le plan d'urbanisme de votre code : elle définit comment les quartiers (couches) communiquent entre eux, où passent les routes (APIs), qui décide des règles (domaine métier) et qui exécute les ordres (infrastructure). Un bon plan permet d'étendre la ville sans tout démolir.

Ce hub couvre les **fondamentaux de l'architecture logicielle** appliqués au développement web moderne.
