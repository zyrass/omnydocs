## Analyse

Le sujet "Architecture" est matérialisé par 3 fichiers (API REST, Clean Architecture, Design Patterns) qui, bien qu'excellents dans leurs thématiques, sont aujourd'hui de simples fichiers temporaires vides (200 bytes).

**Recommandations :**

- Ces concepts (REST, Clean Architecture, Patterns) sont déjà souvent abordés dans les framework-specifics (Vapor, Laravel, Python/Django).
- Soit cette section devient un "Hub" théorique de haut niveau agnostique au langage, soit il faut fusionner ce contenu futur à l'intérieur des cours de POO et Frameworks existants. La création demandera un réel effort de vulgarisation.
