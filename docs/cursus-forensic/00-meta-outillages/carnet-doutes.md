# Carnet de Doutes

!!! abstract "Pourquoi un carnet de doutes ?"
    
    Dans l'investigation numérique, l'incertitude est la norme. Noter ce que l'on ne comprend pas n'est pas un aveu de faiblesse, c'est le début d'une démarche scientifique de résolution.

## Méthode de saisie

Pour chaque blocage, notez :

1. **La date et le contexte**
2. **Le comportement attendu** vs **Le comportement observé**
3. **Les hypothèses** (Pourquoi cela échoue ?)
4. **La résolution** (Une fois trouvée)

## Conclusion

!!! quote "L'art de douter"
    
    > Un bon forensicien est quelqu'un qui documente ses échecs pour construire ses futures réussites.
