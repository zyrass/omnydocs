# I - Législation

!!! quote "L'analogie du chirurgien et du scalpel"

    Un chirurgien manipule chaque jour des outils qui, dans d'autres mains, deviennent des armes. Ce qui distingue le chirurgien du criminel n'est pas l'outil mais le cadre. Le mandat opératoire, le consentement éclairé du patient, le serment d'Hippocrate, la responsabilité civile et pénale. Sans ce cadre, le scalpel devient agression. Avec ce cadre, le scalpel sauve. Le forensic numérique fonctionne exactement de la même manière. Vos outils d'acquisition mémoire, vos techniques de carving, vos exploits de pentest sont neutres. C'est le cadre légal qui transforme votre pratique en métier reconnu plutôt qu'en infraction pénale. Ce module vous donne ce cadre.

## Présentation du module

Ce module pose les fondations juridiques **indispensables** à toute pratique du forensic numérique en France. Il représente 38 heures réparties sur 16 chapitres, qui doivent être traités **avant toute manipulation offensive ou défensive** sur du matériel physique.

### Pourquoi ce module en premier

Trois raisons rendent ce module non négociable et prioritaire.

**Raison 1 - Protection de votre carrière.** Le forensic est l'un des rares métiers où une seule erreur juridique peut entraîner une responsabilité pénale personnelle, indépendamment de votre employeur. Connaître le cadre n'est pas une option, c'est une assurance professionnelle.

**Raison 2 - Crédibilité face aux clients.** Les dirigeants de PME, les magistrats, les avocats vous évaluent sur votre maîtrise du droit avant votre maîtrise technique. Un consultant qui hésite sur l'article 323-1 inspire moins confiance qu'un consultant qui le récite.

**Raison 3 - Recevabilité des preuves.** Une procédure d'acquisition techniquement parfaite peut être annulée si le cadre légal n'a pas été respecté. La rigueur juridique conditionne la valeur probante de votre travail.

### Objectifs pédagogiques

À l'issue de ce module, vous serez en mesure de :

- Citer de mémoire les articles 323-1 à 323-7 du Code pénal et les peines associées.
- Identifier les obligations imposées par le RGPD (articles 32, 33, 34) en cas de violation de données.
- Distinguer un mandat de pentest valide d'un mandat insuffisant ou abusif.
- Rédiger un modèle de mandat de pentest et un NDA exploitables professionnellement.
- Analyser une affaire jurisprudentielle (Bluetouff, Kitetoa) et en tirer des enseignements opérationnels.
- Connaître l'état de transposition de NIS2 en France via la Loi Résilience.
- Tenir un carnet juridique permanent comme référence personnelle.

### Démarrage

Commencez par le chapitre 1.1 (Hiérarchie des normes en France). Aucune manipulation matérielle n'est requise pour ce module. Tout peut être traité pendant que vous attendez la livraison du matériel labo.

!!! warning "Avertissement sur les sources"

    Le droit français évolue. Les articles, peines et procédures décrits dans ce module sont à jour au 28 avril 2026. Vérifiez systématiquement sur Légifrance (legifrance.gouv.fr) avant toute citation officielle dans un rapport client. Le module 1.16 (carnet juridique permanent) inclut une procédure de mise à jour annuelle.
