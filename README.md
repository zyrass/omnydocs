# OmnyDocs

Projet utilisant **Zensical**

### Voir & mettre à jour la version de zensical

_[source](https://zensical.org/docs/upgrade/#how-to-upgrade)_

```bash
# Connaître la version utiliser :
pip show zensical

# Forcer une mise à jour
pip install --uprade --force-reinstall zensical
```
